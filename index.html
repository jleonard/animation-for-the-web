Introduction

There are many techniques for creating animations for the web. We're going to focus on 2 methods for making css based animations. css animations are particularly useful when you want to animate user interfaces (as opposed to making games, animated data visualizations etc.).

Recap

You know what .css and .html are don't you? The quickest way to explain is that html is structure and css is style.

Here's some sample html. It's structred content.

<h1>I'm a headline</h1>
<p>I'm a paragraph. Lorem ipsum dolor...</p>

...and here are css rules that apply style to the structure.

h1{
  font-size: 80px;
  font-family: Arial;
  color: rgb(20,20,20);
}
p{
  font-size: 12px;
  font-family: Georgia;
}

...and if that's too complicated

.html .html + .css

Get it? Let's animate.
